# SmartHome Cortex: AI-Driven Digital Twin Portal | React, Flask, MySQL, Three.js, ESP32

- **Developed a comprehensive Smart Home Ecosystem** integrating a RAG-based (Retrieval-Augmented Generation) AI assistant for conversational device control and real-time sensor data querying.
- **Built an immersive 3D Spatial Digital Twin** using Three.js to visualize energy consumption and device statuses, featuring dynamic room-based camera navigation and interactive power-optimization dashboards.
- **Implemented a secure IoT data pipeline** transmitting sensor data from ESP32 hardware to a MySQL backend, with robust user authentication and role-based access control to safeguard private home automation data.
